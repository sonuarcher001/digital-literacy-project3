## Project Overview
This repository contains my Digital Literacy Project submitted as part of the CSE0001 course at VIT Bhopal University. The aim of this project is to develop awareness about digital literacy, professional online presence, coding platforms, email etiquette, and cybercrime prevention.

The project consists of five tasks that demonstrate essential digital skills required by students in the modern digital world.

# Project Tasks

## Task 1 – Digital Literacy Awareness Infographic
In this task, I created an infographic explaining the concept of digital literacy and its importance for students. The infographic highlights safe internet practices, useful digital tools, and maintaining a professional online presence.
